# V2 Enhancement

> V1 안정화 이후 착수하는 고도화 기능 — 다이내믹 키워드, 임베딩 중복 감지, 선호 프로파일, 대시보드.

---

## V2 범위

> [!note]
> V1 안정화 이후 착수. 착수 게이트 조건을 모두 충족해야 진행한다.

- [ ] 다이내믹 키워드 자동 생성 파이프라인
- [ ] 임베딩 기반 중복 감지 (==코사인 유사도 0.85 기준==)
- [ ] 사용자 선호 프로파일 생성 및 추천 알고리즘 고도화
- [ ] 피드백 루프 분석 대시보드
- [ ] 키워드 사용 통계 시각화

---

## 착수 게이트

| 게이트 | 기준 |
|---|---|
| 루틴 정착 | 최근 4주 기준 주간 발산 세션 ==4회 이상== |
| 수렴 전환 | 최근 4주 기준 월간 Deep Report 4회 이상 |
| 품질 안정 | 중복 아이디어 비율 15% 미만 |
| 실행 연결 | 최소 1개 아이디어가 `실행` 또는 외부 검증 단계로 이동 |

> [!important]
> 4개 게이트 중 하나라도 미충족이면 V2 착수를 보류하고 V1 운영 개선에 집중한다.

---

## 기술 검토 사항

| 기능 | 필요 기술 | 비고 |
|---|---|---|
| 다이내믹 키워드 | 뉴스 크롤링, 키워드 추출, 스케줄러 | Vercel Cron 또는 외부 스케줄러 |
| 임베딩 기반 중복 감지 | OpenAI Embeddings API, 벡터 유사도 | ==비용 추가 발생== |
| 선호 프로파일 | 세션 로그 분석, 통계 모델 | sessions 컬렉션 데이터 활용 |
| 대시보드 | 차트 라이브러리 | Recharts 또는 Chart.js |
