# Target Audience

> 개인용 아이디어 발산/검토 도구. Solo 프로젝트.

---

## 타겟 사용자

| 항목 | 결정 |
|---|---|
| 타겟 사용자 | 본인 (AI 엔지니어) |
| 운영 성격 | Solo 프로젝트, 개인 생산성 도구 |

---

## 사용 패턴

| 시간대 | 세션 | 소요 시간 |
|---|---|---|
| 매일 아침 | 발산 세션 | 5분 |
| 저녁/주말 | 수렴 세션 | 선택적 |

---

## 주요 사용 시나리오

1. 매일 아침 추천 조합 또는 직접 선택한 키워드로 아이디어 10개를 발산한다
2. 그중 북마크한 아이디어를 저녁이나 주말에 다시 꺼내 Deep Report와 평가를 진행한다
3. 결과를 DB에 누적하여 반복 패턴, 선호 키워드, 검토 이력을 남긴다

---

## 제약 조건

> [!important] 개인용 우선
> 멀티 유저나 협업 기능보다 ==개인 루틴 정착==을 먼저 검증한다.

- **세션 분리:** 발산과 수렴을 같은 세션에 몰아넣지 않는다
- **비용 제한:** 월간 API 예산 상한과 세션 횟수 제한을 둔다
- **검증 후 확장:** 다이내믹 키워드, 임베딩, 개인화 추천은 V1 안정화 후 진행한다
