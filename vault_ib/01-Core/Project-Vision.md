# Project Vision

> ==매일 아침 5분, 체계적 키워드 조합으로 비즈니스 아이디어를 발산하고 축적하는 개인 생산성 엔진==

---

## 비전

Idea Bank는 AI 엔지니어가 매일 새로운 비즈니스 영감을 얻을 수 있도록 돕는 인터랙티브 아이디어 워크벤치다. 고정 키워드 프레임워크와 사용자 커스텀 키워드를 결합하여, 인간의 직관과 AI의 생성 능력이 시너지를 내는 ==Human-in-the-Loop== 시스템을 지향한다.

---

## 해결하려는 문제

| 문제 | 설명 |
|---|---|
| 발산 품질 편차 | 자유형 브레인스토밍은 매번 출발점이 달라 품질이 고르지 않다 |
| 기록/분류 부재 | 괜찮았던 아이디어도 체계적 기록 없이 유실된다 |
| 생성-평가 단절 | 생성과 평가가 분리되어 있어 실행으로 이어지지 않는다 |

> [!tip] 핵심 루프
> `구조적 발산 → 선택 → 평가 → 축적` 루프로 위 문제를 해결한다.

---

## 핵심 차별점

기존 대안(ChatGPT Custom GPTs, Notion AI, IdeaBuddy, Validator AI 등)과의 차이:

- **체계적 키워드 조합:** 5개 카테고리 × 96 키워드 매트릭스 기반 구조적 발산
- **평가 프레임워크 내장:** 생성 → PRD → 평가까지 하나의 파이프라인으로 연결
- **축적과 학습:** 세션 로그와 선택/비선택 결과를 저장해 추천과 회고에 활용

---

## V1이 성공한 상태

- 5분 발산 세션이 주 4회 이상 무리 없이 돌아간다
- 북마크된 아이디어가 Deep Report와 평가까지 실제로 이어진다
- 중복 아이디어가 통제되고, 예전 아이디어를 다시 찾을 수 있다
- 운영 중인 규칙이 다음 버전 구현의 기준 문서 역할을 한다
